Detailed Description

This structure provides information about a chameleon-V2 table.

For Chameleon-V0/V1 devices, the following parameters are fix:

  • busType = CHAMELEONV2_BUSTYPE_WB
  • busId = 0
  • file = ""
  • magicWord = 0xABCD
  • reserved = 0

Detailed Description

This structure provides information about a chameleon-V2 device.

For Chameleon-V0/V1 devices, the following parameters are fix:

  • chaRev = 0
  • busId = 0
  • tableNbr = 1
  • bridgeNbr = 0
  • cpuNbr = 0

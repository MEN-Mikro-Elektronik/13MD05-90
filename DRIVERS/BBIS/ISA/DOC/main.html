13BB02 - BBIS Board Driver for generic ISA Devices

Description

The driver supports memory or i/o access to ISA devices. For each ISA
device one board instance must be used.

Variants

The following driver variants exist:

Variant

Description

ISA

driver for memory access

ISA_IO

driver for i/o access

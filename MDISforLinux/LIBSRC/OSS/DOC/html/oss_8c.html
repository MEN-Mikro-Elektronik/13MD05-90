Initializes the OSS module.

Creates an instance of OSS and returns a handle for that instance.

RTAI notes:
It is assumed that this function is always in the context of standard Linux. If called from an RTAI task, this function may block and therefore violate realtime behaviour.
Parameters:
instName IN name of calling module
ossP OUT contains the created handle for that instance
Returns:
0 on success or ERR_OSS_xxx error code on error

OSS module for Linux/RTAI Documentation

This is the documentation of the MEN OSS module (Operating System Services) for Linux and/or RTAI kernel mode.

Refer to the OSS Common Specification for the plain common specification.

Under Linux, OSS is implemented as a loadable kernel module, named men_oss or men_oss_dbg.

Module Parameters:

  • #oss_pci_slot_devnbrs

Special OSS calls under Linux

  • Initialisation and termination: OSS_Init(), OSS_Exit()
  • RTAI timing mode initialisation: OSS_RtaiTimingInit()
  • interface routines to USR_OSS (should not be called by user): OSS_RtaiMsToCount(), OSS_TaskToSigEntry(), OSS_RtaiSigTaskRegister(), OSS_RtaiSigTaskUnregister()

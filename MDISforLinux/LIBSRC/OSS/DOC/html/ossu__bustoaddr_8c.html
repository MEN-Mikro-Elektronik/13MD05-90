Convert bus address to CPU physical address.

Linux notes:
The functionality of this routine depends on the linux kernel configuration:
  • CONFIG_PCI: Must be enabled to support PCI.
  • CONFIG_MEN_VME_KERNELIF: Must be enabled to support VME. This requires a VME kernel mode interface compatible to MEN's specification.
See also:
OSS_MapPhysToVirtAddr(), OSS_AssignResources()

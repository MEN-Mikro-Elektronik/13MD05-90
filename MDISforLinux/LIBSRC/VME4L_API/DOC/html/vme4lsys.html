Functions to control system controller functionality

VME4L provides the following features for a system controller:

  • Control system controller functionality: VME4L_SysCtrlFunctionGet(), VME4L_SysCtrlFunctionSet()
  • Perform VME reset: VME4L_SysReset()
  • Arbiter functions: VME4L_ArbitrationTimeoutGet()

MDIS Libraries

This is the MEN online documentation to all MDIS4 driver and application libraries, which are not documented in the MDIS manuals.

MDIS Driver Libraries

OSS - Operating System Functions for Linux and RTAI drivers
DBG - Debug Output Functions
DESC - Descriptor Decoder Functions
MBUF - Buffer Manager Functions
PLD - PLD Loader Functions
ID - Module ID Functions

MDIS Application Libraries

MDIS_API - MDIS API Functions for Linux User Space
MDIS_API (RTAI) - MDIS API Functions for RTAI Kernel Space
USR_OSS - User Operating System Functions for Linux User Space
USR_OSS (RTAI) - User Operating System Functions for RTAI applications
USR_UTL - Utiltiy Functions

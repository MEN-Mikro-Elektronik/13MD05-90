int Set_BusSwitch (  int  argc,
char *  argv[],
DEV_HDL *  h,
OSS_HANDLE *  osHdl,
u_int8 *  retCnt
)  [static]
 

Set bus switches (A500) in flash.

The A500 has two bus switches which are initialized from the first two bytes in the flash memory. To set this bus switches this function reads the first sector, modifies the first two bytes to match the new setting, erases the first sector and writes the modified data.

Parameters:
argc IN argument counter
argv IN argument vector
h IN DEV_HDL handle
osHdl IN valid OSS_HANDLE structure
retCnt OUT number of command line parameters used
Returns:
success (number of parameters used from command line) or error (-error)

routine to load the FPGA code into the EM04's chameleon FPGA

This routine (re)loads the FPGA on EM04 with the code specified with fpgaCode.

The FPGA header magic and the FPGA code's XOR checksum is checked

This routine can be called without calling ChameleonInit() before loading.

Since this routine accesses the RCS2 latch, the calling routine must allow (setup MMU or BATs) access to the latch.

OSS_MikroDelayInit() must have been called before. I

Parameters:
osh OSS handle
icNumber A015_PCI2VME_FPGA_IC or A015_PCI2MMOD_FPGA_IC
fpgaCode FPGA header followed by code (generated with fpga_addheader)
Returns:
0=ok, or error number CHAMELEON_xxx

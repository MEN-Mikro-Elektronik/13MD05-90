Chameleon-V0 Functions

Attention: The Chameleon-V0 Functions are deprecated and should notbe used for new software designs!

V0::ChameleonInit() creates an handle for a Chameleon device. This handle must be passed to all further V0 functions. V0::ChameleonTerm() destroys the created handle.
Note: The handle of the V0 functions is different from the handle of the V2 functions.

V0::ChameleonGlobalIdent() retrieves the global information (from the Chameleon table header) of a Chameleon device.

V0::ChameleonUnitIdent() retrieves the information about all Chameleon units of the Chameleon device.

V0::ChameleonUnitFind() searches for a Chameleon unit with the specified characteristic(s).

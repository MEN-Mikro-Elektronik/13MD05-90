routine to load the FPGA code into the EM01's chameleon FPGA

This routine (re)loads the FPGA on EM01 with the code specified with fpgaCode.

The FPGA header magic and the FPGA code's XOR checksum is checked

This routine can be called without calling ChameleonInit() before loading.

Since this routine accesses the CPLD, the calling routine must allow (setup MMU or BATs) access to the CPLD (CS1).

OSS_MikroDelayInit() must have been called before.

Parameters:
osh OSS handle
fpgaCode FPGA header followed by code (generated with fpga_addheader)
pldRegBase register base of PLD (CS1)
Returns:
0=ok, or error number CHAMELEON_xxx

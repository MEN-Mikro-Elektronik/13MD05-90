USR_OSS library for Linux Documentation

for USR_OSS Linux fileset 2.15

This is the documentation of the MEN USR_OSS library (User state Operating System Services) for Linux in user land.

Refer to the USR_OSS Common Specification for the plain common specification.

For Linux, the USR_OSS module is compiled as a static or shared library libusr_oss.a or libusr_oss.so, that is linked to application.

The Linux implementation does not add any specific calls to the USR_OSS library.

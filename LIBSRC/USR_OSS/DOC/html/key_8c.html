int32 UOS_KeyPressed (  void   ) 
 

Check if any key pressed on console.

This function checks if any character is buffered in the stdin path. If so, the char this read and returned as integer value. If not, -1 is returned.

Remarks:
The pressed character shall be echoed to stdout
Returns:
-1 if no key pressed or ASCII code of key (0..255)

Linux notes:
temporarily switches console to raw mode
See also:
UOS_KeyWait

Wait until any key pressed on console.

This function waits until any character could be read from the stdin path.

Remarks:
The pressed character shall be echoed to stdout
Returns:
-1 if read error or ASCII code of key (0..255)

USR_OSS specificiation MDIS5

The MEN USR_OSS module (User space Operating System Services) is an abstraction of the services provided by the operating system in user space.

USR_OSS is mainly used to keep MDIS example programs and tools OS independent, but can be used also in other contexts.

Remarks:
Unless otherwise noted, USR_OSS functions return an error code ERR_UOS_xxx and set the global error number errno, that can be retrieved by UOS_ErrnoGet(). This errno is process/task specific.
The common USR_OSS specification provides the following functional groups:

  • Time related: UOS_Delay(), UOS_MikroDelay(), UOS_MikroDelayInit() UOS_MsecTimerGet(), UOS_MsecTimerResolution()
  • Signal Handling: UOS_SigInit(), UOS_SigExit(), UOS_SigInstall(), UOS_SigRemove(), UOS_SigMask(), UOS_SigUnMask(), UOS_SigWait()
  • User Input: UOS_KeyPressed(), UOS_KeyWait()
  • Random number generation: UOS_Random(), UOS_RandomMap()
  • Lists: UOS_DL_NewList(), UOS_DL_Remove(), UOS_DL_AddTail(), UOS_DL_RemHead()
  • Errno: UOS_ErrnoGet(), UOS_ErrnoSet(), UOS_ErrStringTs(), UOS_ErrString()
  • Version info: UOS_Ident()

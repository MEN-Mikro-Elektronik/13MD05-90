int32 UOS_MikroDelay (  u_int32  usec  ) 
 

Delay execution of a process by using a busy-loop.

Delays the calling process for at least usec by wasting CPU time in a busy loop. It can be called from signal handler context.

Remarks:
usec shall be in range 0..1000000 microseconds.

UOS_MikroDelayInit() must be called before this routine can be used.

Parameters:
usec IN number of microseconds to delay
Returns:
0 on success or ERR_UOS_xxx error code on error. On failure, UOS global errno is set to the error code
See also:
UOS_MikroDelayInit

int32 UOS_MikroDelayInit (  void   ) 
 

Initialisation routine for UOS_MikroDelay().

This routine performs any initialisation required to execute UOS_MikroDelay(). For example, it might calibrate a delay loop counter.

Returns:
0 on success or ERR_UOS_xxx error code on error. On failure, UOS global errno is set to the error code

Linux notes:
Since we don't have a timer that we can use, we try to measure a loop counter three times. The minimum time is used and then doubled to get at least the required delay.

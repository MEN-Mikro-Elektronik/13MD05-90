char* UOS_ErrString (  int32  errCode  ) 
 

Convert UOS error number to static string.

Deprecated:
Use of this function in multithreaded applications or under OSes with global namespace is dangerous, since this function uses and returns a static buffer to print its error messages. Better use thread-safe version UOS_ErrStringTs().
Format of returned string: 
     ERROR (UOS) 0x%04x:  <error description>

If the errCode is not an UOS error code, prints "unknown error" as error description.

Parameters:
errCode IN error code ERR_UOS_xxx
Returns:
pointer to error message (static buffer!)
See also:
UOS_ErrnoGet, UOS_ErrStringTs

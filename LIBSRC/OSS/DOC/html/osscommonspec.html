Common OSS specificiation "MDIS4/2003-0"

The MEN OSS module (Operating System Services) is an abstraction of the services provided by the operating system.

OSS is mainly used in MENs Driver Interface System (MDIS) but can be used also in other contexts. In general, OSS is usable only in kernel mode of the OS (if the OS has a kernel mode).

Most OSS implementations provide a routine to initialize and terminate the OSS library: OSS_Init() and OSS_Exit(). OSS_Init() will return the OSS handle that must be passed to all other OSS functions. If not OSS_Init() exists for your OS, pass NULL as the OSS handle.

The common OSS specification provides the following functional groups.

  • Memory allocation: OSS_MemGet(), OSS_MemFree()
  • Interrupt related: OSS_IrqMaskR(), OSS_IrqRestore()
  • SpinLock related: OSS_SpinLockCreate(), OSS_SpinLockRemove(), OSS_SpinLockAcquire(), OSS_SpinLockRelease()
  • Time related: OSS_Delay(), OSS_MikroDelay(), OSS_TickRateGet(), OSS_TickGet()
  • Semaphores: OSS_SemCreate(), OSS_SemRemove(), OSS_SemWait(), OSS_SemSignal()
  • Signals: OSS_SigCreate(), OSS_SigRemove(), OSS_SigSend(), OSS_SigInfo()
  • Alarms: OSS_AlarmCreate(), OSS_AlarmRemove(), OSS_AlarmSet(), OSS_AlarmClear(), OSS_AlarmMask(), OSS_AlarmRestore()
  • C-library substitutes: OSS_Sprintf(), OSS_Vsprintf(), OSS_StrLen(), OSS_StrCpy(), OSS_StrCmp(), OSS_StrNcmp(), OSS_StrTok(), OSS_MemCopy(), OSS_MemFill()
  • PCI access: OSS_PciGetConfig(), OSS_PciSetConfig()
  • Lists: OSS_DL_NewList(), OSS_DL_Remove(), OSS_DL_AddTail(), OSS_DL_RemHead()
  • Swapping: OSS_Swap16(), OSS_Swap32(), OSS_Swap64(), OSS_SWAP16(), OSS_SWAP32(), OSS_SWAP64()
  • Debugging: OSS_DbgLevelSet(), OSS_DbgLevelGet()
  • Version info: OSS_Ident()

The following functions should not be used by MDIS low level drivers, but can be used by the MDIS/BBIS kernel, BBIS drivers and other drivers:

  • Interrupt translation: OSS_IrqLevelToVector()
  • Time related: OSS_MikroDelayInit()
  • Bus translation and mapping: OSS_BusToPhysAddr(), OSS_MapPhysToVirtAddr(), OSS_AssignResources(), OSS_UnAssignResources(), OSS_UnMapVirtAddr()
  • PCI specific: OSS_PciSlotToPciDevice()
  • VMEbus specific: OSS_MapVmeAddr(), OSS_UnMapVmeAddr
  • ISA bus specific: OSS_IsaGetConfig()

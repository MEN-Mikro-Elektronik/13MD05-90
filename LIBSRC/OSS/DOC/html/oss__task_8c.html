u_int32 OSS_GetPid (  OSS_HANDLE *  oss  ) 
 

Get current task id.

Remarks:
Task ID is invalid if called at interrupt level.

Deprecated:
Don't use it. Not all OSes have a task id.
Parameters:
oss IN OSS handle, returned by OSS_Init()
Returns:
task ID of calling process

Linux notes:
The behaviour of this function depends on the rtMode flag passed to the OSS_Init() function. Either the standard Linux kernel or RTAI implementation is called. For RTAI implementation, see OSS_RtaiGetPid().

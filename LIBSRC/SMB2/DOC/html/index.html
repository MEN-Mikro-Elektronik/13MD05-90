SMB2 Library Documentation

Release-1.18

This is the documentation of the SMBus library. The library provides functions to access diverse SMBus/I2C controllers, depending on the built.

Supported Controllers

The SMBus library supports currently the following controllers:
  • MEN 16Z001_SMB FPGA SMB host controller
  • INTEL ICH SMB host controller
  • MGT5200/MPC85XX I2C controller


Functional Description


General/Initialization

A controller specific smbInit function is called. This function returns an array of function pointers, which provide the SMBus access functions specific to the SMBus/I2C controller. The library also supports some special SMBus protocols such as the Alert Response Protocol.


Overview of all Documents

Common functions

smb2_com.c This file provides a set of functions that are common to all controllers. This functions performs SMB protocol specific tasks such as SMB_ALERT handling, software PEC, emulating the SMBus protocol on an I2C bus, ...

Controller specific functions

To every controller supported there exists an own file, smb2_***.c which provides the controller specific functions to the library.
